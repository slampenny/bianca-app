**Use**: When confirmed breach requires individual notification

**Method**: First class mail (preferred) or email (if patient consented)

## MAILING PROCEDURES

### For <500 Individuals:
1. Print on company letterhead
2. Sign each letter (or use signature block)
3. Mail via USPS first class
4. Keep mailing receipts
5. Document date mailed
6. File copies for 7 years

### For 500+ Individuals:
1. Work with mailing service
2. Track delivery
3. Handle returned mail (try alternate addresses)
4. Substitute notice if >10 returned (website posting + media)
5. Maintain proof of mailing

## ALTERNATIVE: EMAIL NOTIFICATION

**Only if**:
- Patient previously agreed to electronic communication
- You have valid email address on file

**Subject Line**: "Important Security Notice from MyPhoneFriend"

**Content**: Same as letter above (adapt for email format)

**Requirements**:
- Encrypted email preferred
- Plain text also acceptable for breach notification (HIPAA exception)
- Track delivery and opens (if possible)
- Send certified/read receipt if available
- Follow up with mail if no response

---

## SUBSTITUTE NOTICE (>10 Insufficient Addresses)

**Required When**: Cannot reach 10 or more individuals by mail

**Methods**:

**1. Website Posting**:
- Conspicuous posting on homepage
- Remain posted for 90 days minimum
- Include all required information

**2. Major Media Notice**:
- Press release to major media in affected areas
- Include all required information
- Maintain proof of publication

**3. Toll-Free Phone Number**:
- Active for 90 days minimum
- Provide information about incident
- Answer questions
- Log all calls

---

## STATE-SPECIFIC REQUIREMENTS

**Some states have additional requirements**:

**California** (example):
- Notification without unreasonable delay (stricter than HIPAA 60 days)
- Must include specific language about credit reports
- Sample notice to California Attorney General

**Check**: https://www.bakerlaw.com/resources/state-data-breach-laws-comparison

**Consult**: State laws for each affected individual's state
